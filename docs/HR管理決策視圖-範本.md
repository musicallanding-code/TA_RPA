# HR 管理決策視圖・範本

> 依決策 1（四層：決策/預警/健康度/密度）與決策 3（＝人才密度 OS 的月度視圖）建立。
> 用法：每月新增一個 `# YYYYMMDD｜HR 管理決策視圖` 區塊；每個數據點都要能回答「**所以主管該做什麼？**」。
> 護欄：敏感人事**去識別**、缺資料標 **fail-safe**（不杜撰）。

---

## 範本結構（填寫指引）

**頂部 banner**：自動生成時間 / 資料來源 / 去識別狀態 / 草稿待審。

**【1 決策層】**本期需主管拍板/關注 — 每項 4 欄：
`問題` ｜ `關鍵數據` ｜ `建議行動` ｜ `不行動的代價`

**【2 預警層】**紅綠燈：🔴 立即處理 / 🟡 注意 / 🟢 良好 / ⚪ 待資料。

**【3 健康度層】**招募漏斗：開缺→pipeline→面試→offer→報到、time-to-fill、補缺結構、瓶頸。

**【4 密度層】**（接 OS）：人才密度指數、BU heatmap、壞的離職趨勢、進用品質。

**（附錄）**OKR/KR 活動明細、補缺清單（去識別）。
