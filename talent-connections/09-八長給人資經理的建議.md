# 09・八長給人資經理的具體建議

每位 C-level 給人資經理(Leo)一句「明天就能做」的具體建議,作為整個人才連結系列的收尾。

| 長 | 建議 | 明天就做 |
|---|---|---|
| **CHRO** | 你最稀缺的是「誠實＋可教練」,別弄丟 | 1-on-1 開頭講「上週你點我 ___,我這週改了 ___」,把回饋變行為 |
| **COO** | 解別人的痛,不是你的痛 | 流程自動化(Kelly/Rita)動工前,先訪談「被自動化的人」,確認他也變輕鬆 |
| **CTO** | 底層買、差異化才自建;上線前自己跑過情境 | 座位系統上線前,交出一張「我 run 過的 edge case 清單」 |
| **CDO** | 先聚焦,再談生態 | 5 案釘成「主推 2＋其餘延後」;MCP/AI Agent 當受控長線、不列本季 KPI;深掘前先過「機率×代價×能否設計消除」 |
| **CFO** | 每個提案配一個能改決策的數字 | 自動化試點先估「省多少工時/成本、回收多久」;用 cost per quality hire 說話 |
| **CIO** | 碰數據,先問同意與權限 | 串接人事資料庫/收資料前確認 PDPA 邊界與權限分級;人才樞紐走 opt-in、不爬履歷 |
| **CMO** | 真實打敗包裝 | Talent Connections 聚焦「他為什麼賭信譽推薦這個人」的真實故事,去金錢化、保留真實挑戰 |
| **幕僚長** | 把「想得大、又落得地」變習慣 | 講戰略命題前先過證據門檻;沒到位就用務實版;懂得「不做什麼」也是高度 |

## 一句話總結
保持你的**誠實與學習力**(贏的底氣);補上**聚焦、嚴謹、同理心**(升級的關卡)。
已從「會出專欄」進步到「想建網絡」——接下來不是做更多,而是把**對的少數**做到讓全經營層信服。
