## 如何召開圓桌會議
本 repo `.claude/skills/cmoney-board/SKILL.md` 已定義八長董事會。
在本 repo 目錄開 Claude Code → `/cmoney-board` → 下「全員圓桌:<議題>」。

> 註:部分數字為公開資訊近似值或內部設定(450 人、組織治理),使用前請以實際資料校正。
