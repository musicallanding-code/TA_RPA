# 教練式 1-on-1 引導腳本(CHRO 用)

原則:**陪跑不審判、幫他聚焦、問出卡點而非考他、看成長斜率。** 建議 45–60 分。

## 會前自我定調
今天目標不是驗收進度,是:幫他收斂 5 案、找出卡點、看他這週比上週長了沒。心法:「我是來幫你把局做小、把路打通的,不是來考你的。」少給答案、多問為什麼。

## 流程
**開場(3 分)・卸壓**
>「你才滿月就扛 5 個案子,這個量是我們派多了,不是你的問題。今天一起把優先序理出來、看哪裡卡住我能幫忙。」
❌ 別一坐下就問「座位系統驗收好了沒」。

**段落1(12 分)・讓他自己排序 → 收斂成 主推2＋維持1＋延後2**
1.「只能全力做 2 件,你選哪 2?為什麼?」
2.「哪一件做好了會讓其他幾件更省力?」
3.「哪一件可以先延後或先做最小版本?」
🟢 能講取捨邏輯、連點成線。🔴「都很重要」迴避取捨。

**段落2(12 分)・問卡點而非考進度**
1.「哪一個讓你最卡、最睡不著?卡在哪?」
2.「有沒有需要我或其他部門幫你擋掉/打通的?」
❌ 聽到卡點別急著給解法,先問「你自己想過哪些做法?」

**段落3(8 分)・驗收當「學習檢核」**
>「座位系統要驗收,我不看它會不會動,想聽你自己 run 過哪些情境?」
1.「除了上次的鎖定/備用座/WFH/交接重疊,你有沒有發現新的?」
🟢 主動列出新發現情境。🔴 還是只證明功能會動。

**段落4(8 分)・回扣上次回饋(驗可教練性)**
1.「上次聊到『不要把痛點轉嫁給一線』,這週你哪裡做了不一樣?」
🟢 有具體行為改變。🔴 答不出上次聊了什麼。

**段落5(5 分)・對齊期待、卸壓**
>「manager 通常 3–6 個月才進入穩定產出。這一兩個月我要你做到『聚焦』和『把基礎做扎實』,不是『開創性大爆發』。」

**收尾(3 分)**
共識:2 主推、1 個我會幫你擋的阻力、下次觀察焦點。結尾肯定誠實＋學得快。

## 速查小抄
| 段落 | 一句話該問 |
|---|---|
| 開場 | 「量是我們派多了,一起理優先序、找卡點。」 |
| 排序 | 「只能做 2 件選哪 2?哪件做好讓其他更省力?」 |
| 卡點 | 「哪件最卡?需要我擋掉什麼?」 |
| 驗收 | 「你自己 run 出哪些情境?有沒有新發現?」 |
| 回饋 | 「上次說的 X,這週做了什麼不一樣?」 |
| 期待 | 「這兩月要聚焦＋基礎,不是開創爆發。」 |

🚫 自我提醒:別變驗收會、別急著給答案、別再加碼新任務。
